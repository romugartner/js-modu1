Instrucciones para la Pre-entrega

Estructura de archivos: Se Crea una carpeta con archivo  index.html  y un archivo  main.js  en su correspondiente carpeta. Vinculo el script en la etiqueta head del HTML con el atributo defer.

Definición del Problema: Situate en una parte del simulador elegido a desarrollar que requiera repetición y decisiones (ej: un contador de puntos de un juego, un validador de stock, o un calculador de promedio de notas).

Implementación de Lógica:

Declare las variables necesarias (const  y  let ).

Cree un bucle para interactuar con el usuario.

Usar un  if/else  para evaluar esos datos. Si la estructura es extensa debe mutar a switch.

Salida de Resultados: Muestra por consola o mediante un alert toda comunicación necesaria para el flujo del script.

Subi los archivos a un repositorio de GitHub y comparti la siguiente URL: https://github.com/romugartner/js-modu1.git
